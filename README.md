## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+
- Firebase account

### Installation

```bash
# Clone the repo
git clone https://github.com/Tusharx1143/KohliVerse.git
cd KohliVerse

# Install dependencies
npm install

# Set up environment variables
```
